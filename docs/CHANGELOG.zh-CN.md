# 更新日志

## 更新日志模板

本文件是 Squeaky Ratkin 的简体中文标准更新日志。

规则：
- 按时间顺序记录，旧版本在前，新版本在后。
- 标题使用本地发布时间：`[YYYY-MM-DD HH:MM UTC+8] 版本 X.Y.Z`。
- Steam 创意工坊首次上传使用 `创意工坊首次上传`，不写成普通版本号。
- 每条改动保持简短清楚，一条 bullet 只写一个改动。
- 需要时分为新增、变更、修复、打包、说明等小节。
- Bugfix 简要带过；只有影响玩家行为时才展开说明。
- 只有同时影响 GitHub 和 Steam 发布面时，才同时提到二者。
- 不记录未完成计划；待办事项写入 `TODO.md`。
- 英文版需同步维护在 `docs/CHANGELOG.md`。

推荐条目格式：

```text
## [YYYY-MM-DD HH:MM UTC+8] 版本 X.Y.Z

一句话概述本次发布。

### 新增
- ...

### 变更
- ...

### 修复
- ...

### 打包
- ...
```

## [2026-07-04 18:35 UTC+8] 版本 0.1.0

首个公开发布版本。

### 新增
- 加入完整的数据驱动鼠族啁啾系统。
- 加入空闲、进食、睡眠、受击、选中、移动、社交、娱乐与死亡触发。
- 加入动作冷却与每个 pawn 的全局冷却控制。
- 加入按心情运行时调制音高与音量。
- 加入原版豚鼠音效兜底。
- 加入自定义音频文件夹支持。
- 加入纯自定义音频模式。
- 加入距离音量衰减预设与自定义衰减区间。
- 加入倍速冷却补偿，用于降低高倍速下的触发密度。
- 加入按语言能力缩放触发频率。
- 加入死亡反馈。
- 加入心情调制工作台，支持精确输入、预设与预览。
- 加入 DebugAction 悬浮字开关与摄像机高度指示器。
- 加入英文与简体中文本地化。
- 加入 Dev、GitHub、Steam 构建 flavor。
- 加入本地测试、GitHub Release 与 Steam staging 打包脚本。

### 修复
- 修复启动 patch 绑定问题。
- 修复 XML range 解析问题。
- 修复 mote 定义红字。
- 修复高倍速音量变小。
- 清理发布包内容。

## [2026-07-05 08:14 UTC+8] 版本 0.1.1

发布构建排障工具与本地化体验修复版本。

### 变更
- 移除了面向玩家排障的 DebugAction 上错误的 Dev-only 编译门。
- DebugAction 现在会随 GitHub 与 Steam 构建一起发布。
- 悬浮字开关与摄像机高度指示器仍然需要 RimWorld 开发者模式与地图内状态。
- 新增游戏内模组名称与简介本地化支持。
- About.xml 保持为英文元数据 fallback。
- 将距离设置标题改为“距离音量衰减”。
- README 新增 3A 声明：AI 设计、AI 开发、AI 绘制。
- 自定义音频权利改由音频提供者自行声明，而不是由本模组统一声明。

### 修复
- 修复发布构建中 DebugAction 缺失的问题。
- 修复模组元数据本地化在语言系统未初始化时触发的翻译红字。
- 移除 release DLL 中的本机调试路径信息。
